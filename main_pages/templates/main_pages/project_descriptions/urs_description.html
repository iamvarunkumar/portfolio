Project Details

The User Risk Scorecard (URS) project focused on developing and automating an end-to-end pipeline to assess user risk within the organization. This involved:

  • **End-to-End Pipeline Development:** Developed and automated a complete user risk scorecard pipeline using Dataiku. This involved orchestrating the entire process from data integration to final reporting.
  • **Data Integration:** Integrated six different security sources into Snowflake, creating a centralized data warehouse for risk analysis.
  • **Initial Scoring Implementation:** Implemented initial scoring mechanisms using Exponential Moving Average (EMA) and Z-scores. This improved the baseline risk detection by 40%.
  • **Predictive Modeling:** Enhanced proactive high-risk user identification by 25% by implementing predictive machine learning models, specifically LightGBM, on Azure ML Services and Docker.
  • **Anomaly Detection:** Integrated unsupervised anomaly detection using Isolation Forest to uncover emerging threats, leading to the identification of 15% more threats.
  • **LLM for Contextual Analysis:** Leveraged Large Language Models (LLMs) through Azure OpenAI for context-aware analysis of alerts, reducing false positives by 20%.
  • **Automation and Deployment:** Fully automated the pipeline, including model retraining and monitoring, resulting in an 80% reduction in manual analysis time. Deployed containerized components using Docker and Azure Functions/ML Endpoints.
  • **Actionable Insights:** Delivered actionable insights through Sigma dashboards, providing clear visualizations of risk data.
